project specifics:
-Components kunnen hun eigen registeren via function pointers en dan wordt
OnOverlapEnter, OnOverlapStay, OnOverlapExit gecalled, met een collderinfo struct
zodat je makkelijk kan vergelijken onder andere met een tag of de rect van de andere collider

-Tilable textures

-Boxcollider size wordt berekent naarmate de texture/sprite + tiling 

-Makkelijk level info kunnen aanpassen in JSON, enemy spawn info, en posities

-Saving scores in binary

-Imgui menus
